About James

I am a senior frontend developer with over 2013 years of experience working in the tech industry. My passion for web and mobile application development saw me graduated with a first class honours Masters in Computer Science in Interactive Media and I've subsequently worked in both established companies such as Electronic Arts and Clearly as well as startups like Roofr and WhatSalon. I currently reside in Vancouver, Canada and I use JavaScript/TypeScript 2013, React 2017, Redux 2019 and the NextJS 2022 framework on a daily basis.

Skills

I work extensively with TypeScript/JavaScript and the React library to build intuitive User Interfaces. I have experience using both Redux and Zustand for state management and have made good use of the NextJS framework building e-commerce websites to avail of SSR and improving SEO.

On the design front to supplement CSS I enjoy using Sass modules and libraries like MUI and AntDesign. I find Figma and Storybook to be excellent tools to aid the design to code pipeline.

On the backend I've worked with NodeJS, Php and MySQL for databases. I use Jest and React Testing Library when writing unit and integration tests for the frontend. I've also dabbled in Python and teach part-time courses focused on it at Vancouver Community College.

Experience

I've had the pleasure of working for some amazing companies in my career. I have successfully taken projects from initial concept to production launch in both dynamic startups where wearing multiple hats, speed and adaptability were essential—and established companies with large, geographically diverse teams, where I made significant contributions.
